LineSensorModel

class LineSensorModel : public SensorModel

Public Functions

inline virtual void update() override

Update object with sensor readings, this function is called every frame (~10ms) and should be used to stash expensive, one time computations before all the particles are calculated, this always runs before p(x) each frame.

inline virtual std::optional<double> p(const Eigen::Vector3f &x) override

Determine p(z_k, x_k). This is the chance that this sensor reading (z_k) would occur given the robot would be at that particle (x_k).

Parameters:

x – The current particle’s position

Returns:

Result of p(z_k, x_k)
